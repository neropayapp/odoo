NeroPay Payment Gateway for Odoo

NeroPay is a secure and reliable payment gateway that allows you to accept online payments directly from your Odoo website, eCommerce, and invoices.

Features

  • ✔️ Easy integration with Odoo Payment Acquirer Framework
  • ✔️ Sandbox and Live environments supported
  • ✔️ Instant Payment Notification (IPN) for auto validation
  • ✔️ Supports currencies: GBP, EUR, IDR

How It Works

  1. Install the NeroPay module in Odoo.
  2. Enter your Public Key and Secret Key in configuration.
  3. Enable and test using sandbox mode.
